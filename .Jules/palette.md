## 2024-03-24 - Accessibility standards for Material Symbols\n**Learning:** The project uses Google Material Symbols Outlined (text ligatures) for icons. These must be hidden from screen readers using `aria-hidden='true'`, with parent buttons requiring explicit `aria-label` attributes to ensure accessibility.\n**Action:** Always add `aria-hidden='true'` to `material-symbols-outlined` spans and `aria-label` to icon-only buttons.
